# BrainFuckCInterpreter
BrainFuckのインタプリタ。
C言語で書かれている。
このソースコードをコンパイルして使う。
第1引数の文字列をBrainFuckのコードとして解釈する。
入力をファイルで行う機能はまだついてない、つけるかもわからない。

以下はターミナルでの記述例
```
 $ gcc BrainFuck.c -o bfci
 $ ./bfci "++++++[->++++[->+++<]<]>>.<+++++++[->++++<]>+.+++++++..+++.>++++[->++++[-<<---->>]<]<---. [->++<]>-.<++++++[->++++<]>.+++.------.--------.<<++++[->++++[->----<]<]>>---...[-]++++++++++."
```
